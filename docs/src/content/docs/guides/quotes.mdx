---
title: Quotes
description: Save, search, annotate, tag, copy, and export passages in Plain.
---

Quotes is Plain's small local passage library.

It is meant for the moment when a readable page gives you a sentence, paragraph, list, table, or code block you want to keep without creating an account or sending the page somewhere else.

## Save Quotes

You can save quotes in two ways:

- Select text in a paragraph and choose **Save Quote** from the context menu.
- Use the quote collection button in the toolbar to select one or more readable blocks, then save the selection.

When selected blocks are not adjacent, Plain keeps the gap visible with `[...]`.

## Review Quotes

Open Quotes from the toolbar or with `Cmd-Option-Q`.

The Quotes view lets you:

- search quote text, notes, tags, source titles, and source URLs
- copy a quote as Markdown
- add a local note
- add local tags
- open the source page in Plain
- remove individual quotes
- export all quotes as Markdown

## Local Storage

Quotes, notes, and tags are stored locally in macOS user defaults.

Plain does not sync Quotes, create an account, or send saved passages to a remote service.
