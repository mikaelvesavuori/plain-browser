# Plain Site

Static single-page marketing site for Plain.

Open `index.html` directly in a browser, or serve the folder from any static host. The assumed production URL is `https://browseplain.com/`.

Important files:

- `index.html` - page content, metadata, OpenGraph tags, and JSON-LD
- `styles/styles.css` - no-framework styling
- `robots.txt` and `sitemap.xml` - crawler entry points
- `llms.txt` and `ai.txt` - AI crawler context
